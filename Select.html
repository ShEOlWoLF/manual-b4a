Select (Switch)

Introducción


La instrucción o select, controla el flujo de ejecución del programa.
Dependiendo de una selección entre múltiples “casos”, ejecuta una secuencia de instrucciones o no las ejecuta.
La Selección es un dato.
Los Casos son valores que pueden contener dicho dato seleccionado.
Dependiendo del valor del dato seleccionado ejecutará una acción.

Ahora viene la programación


Deberemos crea la variable texto para usar en el cuadro de texto esto se hace de la siguiente manera:
Dim Texto As String
Deberemos declarar en el apartado de botón que cuando le demos click esta reproducirá la accion programada derivada del cuadro del texto, es decir que el botón será igual a lo que escribas en el cuadro.
la línea de texto será así: Texto = CuadroTexto.text
Ahora usaremos esto para el Select que sería así:
'Select Texto' esto deberá decir que para activar el comando Select este funcionara con el texto del cuadro de texto
'Case "a"' esto quiere decir que en el caso "a" sucederá lo que escribamos en la siguiente linea.
pondremos Etiqueta.text = "La letra es A" esto con el fin de que cuando selecciones la letra "a" la etiqueta que pusimos en el diseñador muestre la frase antes mencionada.
