Variables

VARIABLES Y TIPOS DE DATOS


Una variable es un espacio de la memoria del ordenador a la que asignamos un valor de algún tipo de dato.
Un tipo de datos se define como un conjunto de valores y las operaciones permitidas en esos valores.

NUMEROS


A continuación veremos la variable byte:
Para la variable byte tenemos un rango de -128 a 127
igual que en el anterior declaramos con dim, el nombre de la variable, as y por último la declaración "byte"
_______________________________________________________________
Para esta declaración a la hora de declarar el valor ya no pondremos comillas como en la pasada, pero recuerda que debes poner el nombre de tu variable y que tienes un límite de números en un rango de -128 a 127, porque en dado caso de salir del rango el programa no compilara.


A continuación veremos la variable short:
Para esta variable tenemos el rango -32768 a 32767
igual que en todas las anteriores declaramos con dim, el nombre de la variable, as y por último la declaración "short"
_______________________________________________________________
Para declarar la variable es el mismo procedimiento del anterior ya que en los numero no se ponen comillas, pero recuerda que debes poner el nombre de tu variable y que solo debemos tener precaución de no sobre pasar los valores "-32768 a 32767"

ADVERTENCIA

En caso de no hacer caso a las indicaciones de valores soportados por variable el programa no se compilará y no se ejecutará.


A continuación veremos la variable long:
Para esta variable tenemos el rango -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807
igual que en todas las anteriores declaramos con dim, el nombre de la variable, as y por último la declaración "long"
Este principalmente lo veremos cuando tengamos números grandes.
_________________________________________________________________
Para declarar la variable de la misma manera que en todas y recuerda que debes poner el nombre de tu variable (sin comillas)


A continuación veremos la variable float:
Para esta variable tenemos el rango +/-3.e+/-38 (menos de 7 dígitos)
igual que en todas las anteriores declaramos con dim, el nombre de la variable, as y por último la declaración "float"
Este es de los más importantes ya que con esta veremos lo que son los puntos decimales y con ello llevaremos mejor conteo a la hora de tener que hacer cálculos.
_________________________________________________________________
Para declarar la variable la misma manera que en todas sin comillas, pero recuerda que debes poner el nombre de tu variable.


A continuación veremos la variable double:
Para esta variable tenemos el rango +/-1.7e+/-308 (menos de 15 dígitos)
igual que en todas las anteriores declaramos con dim, el nombre de la variable, as y por último la declaración "double"
(Esta es para números exageradamente largos)
_________________________________________________________________
Para declarar la variable no debemos pasar de las cantidades dichas y recuerda que debes poner el nombre de tu variable


A continuación veremos la variable boolean:
Para esta variable deberemos declarar con Es false o true (falso o verdadero)
________________________________________________________________
Para esta variable no podremos contestar nada más que true o false, en caso de no darle uno de estos valores no nos compilara y no se ejecutara, tener cuidado de no escribir otra cosa.
